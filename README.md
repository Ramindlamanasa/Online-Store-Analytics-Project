# Online Store Analytics Dashboard

## Tools Used
- SQL
- Power BI
- Excel

## Project Overview
Analyzed retail sales data using SQL and Power BI to identify customer behavior, sales trends, profitability, and return patterns.

## Key Features
- Advanced SQL queries using joins, CTEs, subqueries, and window functions
- Interactive Power BI dashboard
- Business insights and KPI tracking
- Sales and profit analysis

## Key Insights
- Analyzed $2.3M sales data
- Identified top-performing regions and categories
- Found highest return rate areas
- Improved business decision-making using dashboard visualizations

## Files Included
- SQL Queries (.sql)
- Power BI Dashboard (.pbix)
- Presentation (.pptx)
